Gagal mengambil data
Terjadi kesalahan saat menghubungi Google Sheets.
Sekarang menggunakan data contoh. Untuk koneksi otomatis ke Excel/Google Sheets kamu:
1. Buka Google Sheets → File → Share → Publish to web → pilih sheet → format CSV → Publish → copy URL
2. Buka file HTML ini dengan teks editor → cari CSV_URL di baris paling atas script → ganti dengan URL tadi
3. Upload ulang ke GitHub. Dashboard akan auto-refresh setiap buka halaman. ✅
